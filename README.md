# 🛡️ ArgosSystem Hub

**ArgosSystem Hub** es la aplicación cliente central (Centro de Mando) para la plataforma de videovigilancia descentralizada ArgosSystem. Diseñada para operar sobre redes privadas virtuales (VPN), permite monitorear y gestionar múltiples "Nodos Argos" de forma segura y en tiempo real.

## 🚀 Características Principales
* **Monitoreo en Tiempo Real:** Visualización del estado del almacenamiento y métricas de los nodos remotos.
* **Conexión Encriptada:** Integración nativa con redes privadas (ej. Tailscale) para evitar la exposición de puertos públicos.
* **Gestión de Eventos:** Historial de grabaciones y alertas centralizadas.
* **UI/UX Moderna:** Interfaz construida con las últimas directrices de Material Design para Android.

## 🛠️ Stack Tecnológico
* **Lenguaje:** Kotlin
* **Plataforma:** Android (SDK 24+)
* **Arquitectura:** MVVM / Consumo de API REST (JSON)
* **Redes:** Tailscale / VPN Peer-to-Peer

## ⚙️ Instalación y Uso
1. Descarga la última versión del APK desde la sección de [Releases](../../releases).
2. Asegúrate de estar conectado a la misma red virtual (VPN) que tus Nodos Argos.
3. Configura la IP del nodo en los ajustes de la aplicación.
4. ¡Listo para monitorear!
